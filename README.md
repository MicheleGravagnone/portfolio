# Michele Gravagnone | Software Engineer Portfolio

A high-performance portfolio website built with React and CSS3. The project utilizes a light-mode architectural aesthetic featuring warm wood tones and soft frosting effects.

---

## Design Specifications

* **Visual Theme**: Light-mode oak palette with a focus on legibility and organic warmth.
* **Glassmorphism**: Custom frosted glass components utilizing high-blur filters and white-tinted semi-transparent backdrops.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
* **Animations**: Entrance effects driven by the Intersection Observer API and CSS keyframes for a fluid user experience.

---

## Technical Stack

* **Frontend**: React using Hooks and Functional Components.
* **Styling**: Pure CSS3 utilizing custom variables for global theme management.
* **Icons**: SVG-based iconography for resolution independence and performance.
* **Deployment**: Optimized for Vercel, Netlify, or GitHub Pages.

---

## Project Structure

* **Navbar**: Fixed navigation with scroll-aware styling and a minimalist underline animation.
* **Hero**: Landing section featuring an MCP-themed code snippet and floating technology badges.
* **About**: Professional biography including education details and relocation status.
* **Experience**: Chronological career history featuring a polished badge and bullet layout.
* **Projects**: Interactive grid showcasing academic and personal work with custom hover accents.
* **Skills**: Categorized technical toolbox display using pill-shaped tags.
* **Contact**: Integrated social links and a copy-to-clipboard email utility.

---

## Getting Started

### 1. Clone the repository
```bash
git clone [https://github.com/michelegravagnone/portfolio.git](https://github.com/michelegravagnone/portfolio.git)
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm start
```

### 4. Build for production
```bash
npm run build
```
